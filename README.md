# CBFjs
Cross-browser fingerprinting library written in JavaScript.

## Installation

To use CBFjs, simply include `dist/cbf.min.js`.

CBFjs is also available for download via [npm](https://www.npmjs.com/package/cbfjs).

### npm

```shell
npm install cbfjs
```
